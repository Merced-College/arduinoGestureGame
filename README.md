# 🧠 Gesture-Controlled Game (Arduino + Machine Learning)

In this assignment, you will build a gesture-controlled game using the **Arduino Nano 33 BLE Sense Rev2**, a machine learning model, and a web page.

---

## 🚀 Overview

You will:
- Collect motion data for two gestures: **punch** and **flex**
- Train a machine learning model in Colab
- Deploy the model to your Arduino
- Connect your Arduino to a web page
- Control a game:
  - **Punch → Jump**
  - **Flex → Slide**
